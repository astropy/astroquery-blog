  • Beta Release

    2013/09/18
    Adam Ginsburg (keflavich@gmail.com)

    Astroquery has been released in beta as of September 15th, 2013.

    Astroquery's first beta release coincided with the .Astronomy 5 conference, at which a large group of early adopters have tested and contributed to the code base. Within 2 days of the release, we added a new module (GAMA) and added a new tool to SIMBAD to enable criteria queries.

    Astroquery development will continue rapidly. We have the twitters (@astroquerypy) and an e-mail address (astropy.astroquery@gmail.com). We welcome feedback, including suggestions and contributed code.
